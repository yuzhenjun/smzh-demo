基本多选框添加selectpicker
添加属性：
 $('.selectpicker').selectpicker({
      style: 'btn-info'
  });
  
  或使用;data-style="btn-primary" btn-info btn-success btn-warning btn-danger btn-inverse
  
  添加搜索框：data-live-search="true"
  
 选择个数大于0时显示格式化显示data-selected-text-format="count>0" 
 
 限制选择的个数： data-max-option
 
title:在未选择任何选项时显示 

显示箭头：show-menu-arrow show-tick

使用 col-sm-2设置宽度或者使用data-width="auto" data-width="100px" data-width="75%"

使用disabled属性禁用多选框 使用disabled="disabled"可以使某一选项禁用

设置在显示框显示的个数data-size="auto"自适应 data-size="5"

添加分隔线data-divider="true"

参考网址api：http://silviomoreto.github.io/bootstrap-select/
版本：http://cdnjs.com/libraries/bootstrap-select
